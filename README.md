# 21点游戏

## 游戏规则

1. 游戏开始时，玩家和庄家各获得两张牌，庄家的第二张牌是暗牌
2. 玩家可以选择"要牌"或"停牌"
3. 玩家爆牌（超过21点）则立即输掉游戏
4. 玩家停牌后，庄家开始要牌，直到点数达到17或以上
5. 最终比较双方点数，点数较大者获胜（不超过21点）

## 开发环境运行

```bash
npm install
npm run dev
```
